\# UIU Academic - Data Structures \& Algorithms (DSA)



এই রিপোজিটরিতে আমার \*\*Data Structures \& Algorithms\*\* কোর্সের সব ল্যাব টাস্ক এবং প্র্যাকটিস কোডগুলো আছে।



\## প্রজেক্টের বিষয়বস্তু:

\* \*\*ds1.cpp to ds4.cpp\*\*: বিভিন্ন ডেটা স্ট্রাকচার ইমপ্লিমেন্টেশন (যেমন: Linked List, Stack, Queue)।

\* \*\*test.c\*\*: বিভিন্ন প্রাথমিক লজিক টেস্ট করার জন্য ব্যবহৃত ফাইল।



\## পরিবেশ (Environment):

\* \*\*ভাষা\*\*: C / C++

\* \*\*এডিটর\*\*: Visual Studio Code

\* \*\*সিস্টেম\*\*: Windows 11 / Ubuntu
